\# Halli Santhe



Halli Santhe is a hyper-local marketplace Android app built using Kotlin and Jetpack Compose.



\## Features

\- Add Products

\- Browse Products

\- Category Filtering

\- Search Products

\- Seller Details

\- Firebase Integration

\- Gallery Image Selection



\## Technologies Used

\- Kotlin

\- Jetpack Compose

\- Firebase Firestore

\- Firebase Storage



\## Future Improvements

\- Authentication

\- Online Payments

\- Chat System

\- Order Tracking
